What Is Javascript Event Loop..

* JavaScript is a single-threaded programming language. This means that JavaScript can do only one thing at a single point in time. The JavaScript engine executes a script from the top of the file and works its way down. It creates the execution contexts, pushes, and pops functions onto and off the call stack in the execution phase.
